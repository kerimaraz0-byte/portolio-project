# Full-Stack Portfolio Project

This project is a responsive full-stack portfolio website developed using:

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## Features

- Responsive modern UI
- Dark/Light theme switch
- Dynamic projects section
- Contact form with MySQL database
- Admin dashboard
- Session-based authentication
- PHP backend integration
- Fetch API & AJAX system

## Technologies

HTML, CSS, JavaScript, PHP, MySQL, Fetch API
